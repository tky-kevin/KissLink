# Phase 2 Implementation Plan

## Overview

Phase 2 adds Hilt DI, StateFlow, PeerConnector extraction, and a Domain Layer to KissLink. Each step compiles and runs independently.

**Prerequisite**: Phase 1 complete (IdleTeardownManager, ServiceNotificationHelper, WakeLockManager extracted; WiFi routing fixed).

---

## Execution Order

### Step 0A: Room annotationProcessor ‚Üí kapt
**Risk**: LOW | **Time**: 15 min

Migrate Room compiler from `annotationProcessor` to `kapt` so Kotlin files can be processed.

**Files**:
- `app/build.gradle` ‚Äî add `kotlin-kapt` plugin, replace `annotationProcessor` with `kapt`, add `kapt { arguments { ... } }`

**Verify**: `./gradlew :app:assembleDebug` succeeds.

---

### Step 0B: Add Hilt dependencies
**Risk**: LOW | **Time**: 15 min

**Files**:
- `build.gradle` (root) ‚Äî add `id 'com.google.dagger.hilt.android' version '2.51.1' apply false`
- `app/build.gradle` ‚Äî add Hilt plugin + `implementation 'com.google.dagger:hilt-android:2.51.1'` + `kapt 'com.google.dagger:hilt-android-compiler:2.51.1'`

**Verify**: `./gradlew :app:assembleDebug` succeeds.

### Step 2.3: Extract PeerConnector
**Risk**: MEDIUM | **Time**: 1.5 hrs

**Create**: `app/src/main/java/com/kisslink/transfer/PeerConnector.kt`
- `@Singleton class PeerConnector @Inject constructor()`
- `acceptAsServer(callback)` ‚Äî GO side, ServerSocket with 20s timeout
- `connectAsClient(p2pNetwork, callback)` ‚Äî Client side, 30-attempt retry with `Network.bindSocket()`
- `fun interface Callback { onSocketReady(Socket); onError(String) }`

**Modify**: `FileTransferService.java`
- `@Inject PeerConnector peerConnector`
- `establishPeer()` delegates to `peerConnector.acceptAsServer()` / `peerConnector.connectAsClient()`
- Remove `acceptAsServer()` and `connectAsClient()` methods (~45 lines removed)

**Verify**: TCP connection works for both GO and client paths.

### Step 2.5: Domain Layer (SessionManager + TransferManager)
**Risk**: HIGH | **Time**: 4 hrs

**Create**:
- `app/src/main/java/com/kisslink/domain/SessionManager.kt` ‚Äî owns `StateFlow<SessionState>`, session lifecycle, peer identity, generation guard
- `app/src/main/java/com/kisslink/domain/TransferManager.kt` ‚Äî owns PeerConnection lifecycle, progress tracking, transfer recording

**Modify**: `FileTransferService.java` ‚Äî becomes thin shell (~250 lines):
- `@Inject SessionManager sessionManager`
- `@Inject TransferManager transferManager`
- `createCoordinator()` delegates phase/error to `sessionManager`
- `startPeer()` delegates to `transferManager`
- `teardownPeer()` delegates to `transferManager.close()`
- Binder returns `sessionManager.state` and `transferManager.progress`

**Verify**: Full end-to-end: NFC pairing ‚Üí file transfer ‚Üí history record ‚Üí disconnect.
